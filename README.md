# TI-30xplus USB-C-Mod
A small mod to add a USB-C port for power to the TI-30xplus

The PCB consists if a 5V to 3V transistor, which gets its powert from the USB-C port.
The USB-C port is regulated to 5V via two 5.1K Ohm Resistors.
The two testpoints are used as contact points to atach to the calculator.
